# Module 10 - Secure User Model

This project extends the FastAPI calculator application by adding a secure user authentication model using SQLAlchemy, Pydantic validation, PostgreSQL, Docker, and GitHub Actions.

## Features

- FastAPI web application
- PostgreSQL database
- SQLAlchemy ORM
- Secure password hashing
- Pydantic validation
- Unit and integration testing
- Docker support
- GitHub Actions CI/CD
- Docker Hub deployment
